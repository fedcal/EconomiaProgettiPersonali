## SEO On-Page

### Ottimizzazione Contenuti
- **Title Tag**: 50-60 caratteri, keyword principale all'inizio
- **Meta Description**: 150-160 caratteri, call-to-action chiara
- **Header Tags**: Struttura gerarchica H1 > H2 > H3
- **Keyword Density**: Naturale, evitare keyword stuffing
- **Internal Linking**: Collegare contenuti correlati

### Ottimizzazione Immagini
- Nome file descrittivo (es: casa-vacanze-salento.jpg)
- Alt text ottimizzato
- Formato WebP per performance
- Lazy loading per immagini below the fold
- Dimensioni appropriate (responsive)

### URL Structure
- URL brevi e descrittivi
- Keyword nel percorso
- Evitare parametri inutili
- Struttura gerarchica chiara

## SEO Tecnico

### Core Web Vitals
- **LCP** (Largest Contentful Paint): < 2.5s
- **FID** (First Input Delay): < 100ms
- **CLS** (Cumulative Layout Shift): < 0.1

### Performance
- Minificazione CSS/JS
- Compressione Gzip/Brotli
- Caching strategico
- CDN implementation
- Riduzione HTTP requests

### Mobile-First
- Design responsive
- Touch-friendly elements
- Velocità mobile ottimizzata
- Test su dispositivi reali

### Struttura Tecnica
- Sitemap XML aggiornata
- Robots.txt configurato
- Schema markup (JSON-LD)
- Canonical tags
- HTTPS obbligatorio

## SEO Locale

### Google Business Profile
- Completare tutti i campi
- Foto di qualità (min 10)
- Post settimanali
- Rispondere alle recensioni
- Q&A ottimizzate

### Citations e Directory
- NAP consistency (Name, Address, Phone)
- Directory locali di settore
- Mappe e servizi geo-localizzati

### Local Keywords
- "servizio + città"
- "vicino a me"
- Quartieri e zone specifiche

## Tools e Analytics

### Tools Essenziali
- **Google Search Console**: Performance, copertura, usabilità
- **Google Analytics**: Traffico, comportamento, conversioni
- **Google PageSpeed Insights**: Performance e Core Web Vitals
- **Screaming Frog**: Crawling e audit tecnico
- **Ahrefs/SEMrush**: Keyword research e competitor analysis

### Metriche da Monitorare
- Traffico organico
- Posizionamento keywords
- CTR (Click-Through Rate)
- Bounce rate
- Tempo di permanenza
- Conversioni da organico
- Backlinks (quantità e qualità)
